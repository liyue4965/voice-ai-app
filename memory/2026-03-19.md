## 工业节能模型 - 水泥窑脱硝（SNCR）优化

### 业务场景
水泥窑分解炉脱硝系统氨喷射量优化控制

### 输入变量（7个）
| # | 变量 | 说明 |
|---|------|------|
| 1 | 分解炉温度 | °C |
| 2 | 窑转速 | rpm |
| 3 | 窑喂料量 | t/h |
| 4 | 三次风温 | °C |
| 5 | 风压力 | Pa |
| 6 | 入口NOX含量 | mg/m³ |
| 7 | 出口NOX含量 | mg/m³ |

### 输出变量
- **氨喷射量**（目标优化值）

### 优化目标
1. **最低氨耗** - 减少氨水/尿素消耗成本
2. **排放达标** - 出口 NOx ≤ 50 mg/m³
3. **氨逃逸控制** - ≤ 8 mg/m³
4. **稳定控制** - 避免剧烈波动

### 约束条件
- 出口NOx目标：≤ 50 mg/m³
- 氨逃逸量：≤ 8 mg/m³
