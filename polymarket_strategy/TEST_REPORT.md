## 一、策略框架

### 7 信号动量系统

| # | 信号 | 公式输出 | 说明 |
|---|------|---------|------|
| 1 | s1_rsi | -1/0/+1 | RSI 超买超卖 |
| 2 | s2_macd | -1/0/+1 | MACD 交叉 |
| 3 | s3_bands | -1/0/+1 | 布林带突破 |
| 4 | s4_volume | -1/0/+1 | 成交量确认 |
| 5 | s5_atr | -1/0/+1 | ATR 波动率收缩 |
| 6 | s6_adx | -1/0/+1 | ADX 趋势强度 |
| 7 | s7_mean | -1/0/+1 | 均值回归 |

### 核心公式

```
R_技术 = (R_max / 7) * Σ s_i(t)
R_情绪 = sentiment * R_max
R_最终 = (1 - sentiment_weight) * R_技术 + sentiment_weight * R_情绪
```

## 二、测试改进点

根据文章研究，新增三个改进方向：

1. **动态止盈** - 分批卖出 (5%/10%/15%)
2. **时间衰减因子** - 结算前反向信号增强
3. **流动性筛选** - 筛选中等流动性市场

---

## 三、测试结果

### 3.1 动态止盈 vs 固定止盈

| 策略 | 收益率 | 交易次数 |
|------|--------|---------|
| 固定止盈 (8%) | **-0.28%** | 51 |
| 动态止盈 (8%/15%/25%) | -6.36% | 26 |

**结论:** 固定止盈效果更好，动态止盈容易卖飞错过大趋势

---

### 3.2 时间衰减因子

| 策略 | 收益率 | 交易次数 |
|------|--------|---------|
| 无时间衰减 | **+6.13%** | 57 |
| 有时间衰减 (结算前48h, x1.2) | -3.71% | 43 |

**结论:** 时间衰减因子反而增加亏损，建议关闭

---

### 3.3 流动性筛选

| 市场 | 无筛选 | 有筛选 ($50K-$2M) |
|------|--------|---------|
| BitBoy ($122K) | +0.17% | **+2.68%** ✅ |
| Russia-Ukraine ($1.4M) | -2.33% | **+1.28%** ✅ |
| Fed Rate ($850K) | -3.62% | **-1.69%** ✅ |
| 低量市场 ($30K) | +12.77% | ❌ 被过滤 |

**结论:** 流动性筛选有效，过滤低流动性市场后收益明显改善

---

### 3.4 综合测试

| 配置组合 | 平均收益率 | 交易次数 |
|----------|-----------|---------|
| 原始策略 | - | - |
| V2 (动态止盈+时间衰减+筛选) | +1.75% | 183 |
| V2 (仅筛选，无时间衰减) | +6.13% | 57 |

## 四、最优配置

```python
class BestConfig:
    """Polymarket 7-Signal 最优配置"""
    
    # 风险参数
    R_max = 0.20              # 最大风险敞口
    stop_loss = 0.03          # 止损 3%
    take_profit = 0.08        # 固定止盈 8%
    
    # 信号参数

    signal_threshold = 0.01   # 信号阈值
    
    # 权重
    weights = {
        "rsi": 1.0,
        "macd": 1.0,
        "bollinger": 1.0,
        "volume": 0.8,
        "atr": 0.7,
        "adx": 1.2,
        "mean_reversion": 0.8
    }
    
    # 流动性筛选 (开启)
    min_volume = 50000        # 最小成交量 $50K
    max_volume = 2000000      # 最大成交量 $2M
    
    # 时间衰减 (关闭)
    time_decay_enabled = False
```
