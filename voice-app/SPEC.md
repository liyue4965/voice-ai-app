# VoiceAI 语音转文字 App - 设计规范

## 1. 项目概述

- **项目名称**: VoiceAI 语音转文字
- **类型**: 跨平台 App (iOS + Android + 微信小程序 + Web)
- **框架**: Taro 3.x (React)
- **核心功能**: 实时语音录制 + 语音转文字

## 2. UI 设计（完全参考图片）

### 2.1 整体风格
- **主题**: 深色科技风

- 高度：60px

**字体：**
- 标题：18px, 600 weight
- 正文：14px, 400 weight
- 计时器：32px, 600 weight, 等宽字体

## 3. 功能规范

### 3.1 核心功能
1. **录音** - 点击开始，再次点击停止
2. **实时识别** - 录音时实时显示波形
3. **语音转文字** - 录音结束后显示识别结果
4. **复制** - 一键复制识别结果
5. **历史记录** - 保存每次录音和识别结果

### 3.2 用户交互
- 点击录音按钮 → 开始录音 + 动画开始
- 再次点击 → 停止录音 + 开始识别
- 识别完成 → 显示文字结果
- 点击复制 → 复制到剪贴板
- 点击Tab → 切换页面

### 3.3 数据存储
- 使用本地存储保存历史记录
- 数据格式：JSON
