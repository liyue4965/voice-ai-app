## 一、认知准备

### 1.1 外贸是什么

外贸 = 把国内产品卖给国外客户

**核心逻辑：**
- 国内采购 → 出口报关 → 海运/空运 → 国外客户收货 → 收款

**管件出口特点：**
- 产品重（海运为主）
- 规格多（需详细确认参数）
- 材质要求严格（材质单、检测报告）
- 认证需求（ISO、CE、API等）

### 1.2 外贸 vs 内贸

| 方面 | 内贸 | 外贸 |
|------|------|------|
| 语言 | 中文 | 英语/外语 |
| 付款 | 灵活 | 需考虑汇率、信用证 |
| 物流 | 国内快递 | 海运/空运+报关 |
| 周期 | 短 | 长（海运15-45天） |
| 客户 | 可见 | 需远程开发 |
| 利润 | 较低 | 相对较高 |

### 1.3 做外贸的三种模式

**模式一：SOHO（个人创业）**
- 优点：投入小、风险低
- 缺点：需要全能、信任度低
- 适合：兼职试水、积累经验

**模式二：挂靠外贸公司**
- 优点：借用公司资质、抬头
- 缺点：利润分成、有竞争限制
- 适合：刚起步没有公司

**模式三：自建公司**
- 优点：自主可控、利润独享
- 缺点：投入大、运营复杂
- 适合：有一定资源和资金

## 二、资质办理

### 2.1 需要的证件

| 证件 | 作用 | 办理难度 | 费用 |
|------|------|----------|------|
| 营业执照 | 主体资质 | 简单 | 免费 |
| 进出口经营权 | 可自行出口 | 简单 | 1000-2000 |
| 海关编码 | 报关用 | 简单 | 免费 |
| 电子口岸卡 | 线上报关 | 简单 | 约500 |
| 外汇管理局备案 | 收汇 | 简单 | 免费 |

### 2.2 办理流程

**自己办理（推荐时间充裕时）：**
1. 营业执照增加"货物进出口"经营范围
2. 海关办理进出口收发货人备案
3. 电子口岸办理法人卡和操作员卡
4. 外汇管理局企业备案
5. 税务局出口退税备案

**找代理办理（推荐新手）：**
- 费用：1000-2000元
- 时间：1-2周
- 优点：省事、他们熟悉流程
- 搜索关键词：*"进出口权代办"*

### 2.3 外贸代理公司

如果暂时没有公司或进出口权，可以用外贸代理：

**什么是外贸代理？**
- 代理公司帮你出口，你用他们的公司抬头
- 报关、物流、结汇他们操作
- 你付代理费（通常1-2%）

**如何找外贸代理？**
- 搜索："城市 + 外贸代理公司"
- 阿里1688也有服务商
- 福步论坛里有推荐

**代理合作流程：**
```
1. 签订代理协议
2. 客户提供货描、报价
3. 代理公司抬头报价给国外客户
4. 客户付款到代理账户
5. 代理扣除费用后转给你
6. 你国内采购付款给工厂
```

## 三、产品学习

### 3.1 管件基础知识

**孟村管件主要产品：**

| 产品 | 英文 | 用途 |
|------|------|------|
| 弯头 | Elbow | 改变管道方向 |
| 三通 | Tee | 分支连接 |
| 四通 | Cross | 十字分支 |
| 法兰 | Flange | 连接、密封 |
| 异径管 | Reducer | 变径连接 |
| 管帽 | Cap | 封口 |

### 3.2 规格参数

**需要掌握的关键参数：**

```
1. 公称直径 (DN)
   - DN15, DN20, DN25 ... DN2000
   
2. 壁厚 (Schedule)
   - SCH10, SCH20, SCH40, SCH80, SCH160
   
3. 压力等级 (Pressure Class)
   - PN16, PN25, PN40, PN64
   - Class 150, 300, 600, 900, 1500
   
4. 材质 (Material)
   - 碳钢: A234 WPB, A350 LF2
   - 不锈钢: A403 WP304, WP316
   - 合金钢: A234 WP11, WP22
   
5. 标准 (Standard)
   - GB/T12459, GB/T13401
   - ASME B16.9, B16.28
   - DIN 2605, 2615
   - JIS B2311, B2312
```

### 3.3 材质对比

| 材质 | 代码 | 适用场景 | 特点 |
|------|------|----------|------|
| 碳钢 | CS | 一般工业 | 便宜、强度高 |
| 不锈钢 | SS304/316 | 耐腐蚀 | 贵、不生锈 |
| 合金钢 | WP11/WP22 | 高温高压 | 耐热、耐腐蚀 |
| 铸钢 | WCC | 大口径 | 成本低 |

### 3.4 怎么学习产品

**推荐方法：**
1. **看工厂目录** - 让工厂发产品样本
2. **逛行业网站** - 搜索"pipe fitting manufacturer"
3. **逛阿里国际站** - 看同行怎么描述产品
4. **问工厂技术** - 不懂就问，很重要
5. **跑工厂** - 最好去孟村实地看看

**产品资料准备：**
- [ ] 产品目录（PDF）
- [ ] 价格表（规格对应价格）
- [ ] 材质单样本
- [ ] 检验报告样本
- [ ] 工厂资质证书

## 四、工厂资源

### 4.1 找工厂渠道

**线上：**
- 1688搜索："孟村 管件"
- 阿里国际站找工厂
- 中国制造网

**线下：**
- 去孟村实地考察
- 通过朋友介绍
- 参加行业展会

### 4.2 考察工厂问什么

**基础信息：**
- 你们主要做什么产品？
- 有自营出口权吗？
- 有哪些认证？ISO/API/CE？
- 最小起订量是多少？
- 交货周期多久？

**价格相关：**
- 能给到什么价格？（报规格）
- 价格含税吗？含运费吗？
- 付款方式是什么？

**质量相关：**
- 有什么检测设备？
- 材质证明怎么办？
- 能否提供样品？

### 4.3 合作模式

**模式一：工厂直供**
- 你接单 → 工厂生产 → 你出口
- 优点：价格可控、质量可监
- 缺点：需要垫资

**模式二：代理出口**
- 你接单 → 工厂供货 → 外贸代理出口
- 优点：不用自己出口
- 缺点：利润被分走

**模式三：合作生产**
- 和工厂签长期协议
- 优点：价格稳定、优先排单
- 缺点：需要一定量

### 4.4 首次合作注意事项

1. **小单试水** - 先做小单测试工厂配合度
2. **书面确认** - 所有约定要写下来
3. **样品确认** - 大货前先确认样品
4. **质量检验** - 发货前最好验货
5. **保留凭证** - 沟通记录、合同都要保存

## 五、获客渠道

### 5.1 平台获客

#### 阿里国际站

**入驻费用：**
- 基础会员：29800元/年
- 金品诚企：80000元/年
- 广告费：每月2000-10000元

**操作要点：**
1. 完善公司主页（工厂视频、车间照片）
2. 上传30-50个产品
3. 产品关键词优化（英文）
4. 每天及时回复询盘
5. 适当投入广告（P4P）

**适合产品：** 弯头、法兰、三通等标准件

#### 中国制造网

**入驻费用：**
- 基础会员：19800元/年
- 高级会员：32800元/年

**优点：**
- 门槛比阿里低
- 询盘质量较高
- 适合工业品

#### 独立站 + Google推广

**费用：**
- 建站：3000-10000元
- Google广告：每月5000元起

**优点：**
- 主动权在自己手里
- 积累自己的客户资源

**缺点：**
- 需专业运营
- 效果需要时间

### 5.2 开发信开发

**什么是开发信？**
- 主动给潜在客户发邮件介绍产品

**开发信模板：**

```
Subject: Quality Pipe Fittings from China Manufacturer

Dear [Name],

Hope this email finds you well.

We are [Company Name], a professional pipe fittings manufacturer from Mengcun, China - known as the "Pipe Fitting Capital" of China.

Our main products include:
- Elbows (45°, 90°, 180°)
- Tees (Equal & Reducing)
- Flanges (Weld Neck, Slip-on, Blind)
- Reducers, Caps, etc.

We have ISO 9001, API certifications, and export to Europe, Middle East, Southeast Asia for many years.

Could you please let us know what products you are interested in? We will send you our catalog and competitive quote.

Best regards,
[Your Name]
[Company Name]
Tel: [Phone]
Email: [Email]
```

**发开发信技巧：**
- 不要太长（100-150词）
- 突出产品优势
- 不要有语法错误（检查后再发）
- 建议用专业英文简历网站检查

**客户哪里找：**
- LinkedIn搜索
- Google搜索："pipe fittings importer"
- 海关数据（后面讲）
- 展会名片

### 5.3 海关数据

**是什么？**
- 记录进出口交易数据
- 可以查到谁在进口、进口价格

**作用：**
- 找到目标客户
- 了解市场价格
- 分析竞争对手

**推荐平台：**
- 腾道
- 环球慧思
- 国贸通

**使用方法：**
1. 搜索产品HS编码（如管件：73079300）
2. 筛选目标国家
3. 获取进口商名单
4. 查到联系方式后发开发信

## 六、成交转化

### 6.1 询盘处理流程

```
1. 收到询盘（24小时内回复）
       ↓
2. 分析客户需求（规格、数量、目标价）
       ↓
3. 核算成本（原材料+加工+运费+利润）
       ↓
4. 制作报价单（PDF/Excel）
       ↓
5. 回复报价（带图片、参数表）
       ↓
6. 跟进沟通（客户问价通常3-5轮）
       ↓
7. 确认订单（签订合同）
       ↓
8. 收取定金（30%预付）
       ↓
9. 下单生产
```

### 6.2 报价单模板

```
QUOTATION

Date: [日期]
Quote No.: [报价编号]

From:
[公司名称]
[地址]
[电话/邮箱]

To:
[客户名称]
[客户地址]

Product: [产品名称]
Specification: [规格参数]
Material: [材质]
Standard: [执行标准]
Quantity: [数量]
Price: [价格] USD/Piece
Port of Loading: [起运港]
Payment Term: [付款方式]
Valid Until: [有效期]

Thanks for your inquiry!
[你的名字]
```

### 6.3 合同签订

**合同要素：**
- 产品规格、材质、数量
- 单价、总价
- 付款方式
- 交货期
- 包装要求
- 检验标准
- 违约责任

**合同模板关键条款：**

```
1. 产品描述
2. 数量与价格
3. 付款条件：30% deposit, 70% against B/L copy
4. 交货时间：within 30 days after deposit received
5. 装运港/目的港
6. 包装：standard export packing
7. 检验：inspection before shipment
8. 争议解决：arbitration
```

### 6.4 常见价格术语

| 术语 | 含义 | 卖方责任 | 买方责任 |
|------|------|----------|----------|
| EXW | 工厂交货 | 仅提供货物 | 承担全部费用和风险 |
| FOB | 离岸价 | 运到指定港口 | 运费和保险 |
| CIF | 到岸价 | 运到目的港+保险 | 到港后费用 |
| CFR | 成本加运费 | 运到目的港 | 保险 |

**新人建议：** 先做FOB，熟悉后再做CIF

## 七、物流与收款

### 7.1 发货流程

```
1. 生产完成
       ↓
2. 验货（质量、数量、包装）
       ↓
3. 订舱（货代公司）
       ↓
4. 报关（提供发票、装箱单、合同）
       ↓
5. 装柜（集装箱）
       ↓
6. 提单（B/L）
       ↓
7. 通知客户付款
       ↓
8. 收齐尾款后寄正本提单
```

### 7.2 货代选择

**什么是货代？**
- 帮你安排海运、报关的公司

**怎么找货代：**
- 阿里物流板块
- 百度搜索
- 朋友推荐

**选择要点：**
- 价格合理
- 服务及时
- 有报关资质

### 7.3 收款方式

| 方式 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| T/T电汇 | 银行转账 | 简单、手续费低 | 需先收款 |
| 西联 | Western Union | 快速 | 额度小 |
| PayPal | 国际支付 | 方便 | 手续费高 |
| 信用证L/C | 银行担保 | 安全 | 复杂 |

**新人建议：** 用T/T，30%预付，70%提单COPY

### 7.4 结汇操作

**有进出口权：**
- 自己报关、自己结汇
- 收到外汇后去银行结汇

**用外贸代理：**
- 代理帮你结汇
- 扣除代理费后人民币给你

## 八、新手建议

### 8.1 一定要做到

1. **产品要专一**
   - 先只推1-2款产品
   - 别什么都想做
   - 专而精更容易成交

2. **英语不好也要敢开口**
   - 可以用翻译软件
   - 重点是把产品搞熟
   - 邮件沟通可以慢慢写

3. **从小单开始**
   - 第一单控制在5000-10000美金
   - 积累经验和信心

4. **选择靠谱客户**
   - 新客户预付为主
   - 付款方式宁严勿松

### 8.2 一定要避免

1. **不要垫资**
   - 收到客户款再付工厂
   - 外贸代理帮你操作

2. **不要囤货**
   - 先询盘，询到单再采购
   - 零库存起步

3. **不要贪大**
   - 大单风险大
   - 新手做小单更安全

4. **不要赊账**
   - 宁可不做也不要欠款
   - 坏账会拖垮你

### 8.3 成长路径

**第1-3个月：学习**
- 学产品知识
- 熟悉平台操作
- 了解外贸流程

**第3-6个月：询盘**
- 开始发开发信
- 尝试平台运营
- 积累潜在客户

**第6-12个月：成交**
- 争取第一单
- 积累成功案例
- 建立客户信任

**1年以后：稳定**
- 有了稳定客户
- 优化运营效率
- 考虑团队扩展
