# 工业仿真数字孪生编辑器

Industrial Digital Twin Editor - 基于 Web 的 3D 可视化编辑平台

## 功能特性

### 核心功能
- 🧊 **3D 编辑器** - 创建和编辑工业场景，支持立方体、球体、圆柱体、圆锥体、圆环、平面等几何体
- 💡 **光照系统** - 支持环境光、平行光、点光源
- 🎨 **材质编辑** - PBR 材质、金属度、粗糙度、透明度的实时调整
- 📐 **变换控制** - 移动、旋转、缩放工具
- 🌲 **场景层级** - 树形结构管理场景对象

### 预览模块
- 🖐 **轨道模式** - 鼠标拖拽旋转、缩放、平移
- 👤 **第一人称模式** - WASD 移动，沉浸式浏览
- ⛶ **全屏预览** - 支持全屏查看
- 📊 **实时统计** - FPS、物体数量、面数显示

### 管理中心
- 📁 **项目管理** - 新建、保存、加载项目
- 🖼️ **场景管理** - 内置模板（智能工厂、智慧城市、智慧农业、智能仓储、能源站）
- 📦 **资源管理** - 内置几何体库、材质库，支持自定义资源上传

### 数据与 AI
- 📡 **数据绑定** - 支持 REST API、WebSocket、MQTT 数据源接入
- 🤖 **AI 生成** - 设计图/文字描述生成 3D 场景（需接入 AI API）
- 📤 **导出功能** - 导出为独立 HTML 文件

## 技术栈

- **前端框架**: React 18 + TypeScript
- **3D 引擎**: Three.js
- **状态管理**: Zustand
- **构建工具**: Vite 5

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本

```bash
npm run build
```

## 使用说明

### 1. 创建场景
点击「+ 添加物体」创建几何体或光源

### 2. 编辑对象
- 在左侧场景层级点击选择对象
- 在右侧属性面板调整位置、旋转、缩放
- 使用顶部工具栏切换变换模式

### 3. 预览效果
点击绿色的「▶ 预览」按钮进入预览模式

### 4. 保存与导出
- 点击「💾 保存」下载 JSON 文件
- 点击「📤 导出」生成独立 HTML

### 5. 管理中心
点击「⚙️ 管理中心」管理项目、场景和资源

## 注意事项

- 3D 功能需要浏览器支持 WebGL
- 推荐使用 Chrome、Firefox 或 Edge 浏览器
- 确保显卡驱动已更新
