### 策略框架
用户分享了一个量化+行为模拟的交易框架：

1. **传统动量系统** (Nasdaq 期货)
   - 公式：`R(t) = R_max · m(t) = (R_max / 7) · Σ s_i(t)`
   - 7 个独立动量信号，风险加权叠加
   - 每个信号 s_i(t) ∈ {-1, 0, +1}
   - 信号共振时满仓，分歧时空仓
   - 实盘 $35M+ alpha +1406%

2. **MiroFish 行为模拟**
   - 数字沙盘：成千上万 AI agents 模拟人群行为
   - 给定场景 → 模拟市场情绪扩散方向
   - 捕捉 "human behavior alpha"

3. **核心观点**：两者叠加 = 价格 alpha + 行为 alpha

### 核心公式实现

```python
# 技术信号
R_技术 = (R_max / 7) * Σ s_i(t)

# 情绪信号
R_情绪 = sentiment * R_max

# 组合公式
R_最终 = (1 - sentiment_weight) * R_技术 + sentiment_weight * R_情绪
```
