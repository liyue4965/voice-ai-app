# 进出口权办理完整指南

> 适用于：首次办理进出口权的企业 / 个人SOHO
> 更新时间：2026-03-09

---

## 一、什么是进出口权

进出口权 = "可自行开展进出口业务的资格"

**有了进出口权，你可以：**
- ✅ 用自己公司名义报关出口
- ✅ 自行收汇、结汇
- ✅ 申请出口退税
- ✅ 不用依赖外贸代理

---

## 二、办理进出口权的两种方式

### 方式一：自己办理（免费，但耗时间）

**办理部门与流程：**

```
1. 工商局
   └── 营业执照增加"货物进出口"经营范围
   
2. 海关
   └── 办理"进出口收发货人备案"
   └── 获得：海关编码 + 海关登记证

3. 电子口岸
   └── 办理法人卡、操作员卡
   └── 获得：电子口岸IC卡

4. 外汇管理局
   └── 企业备案登记
   └── 获得：外汇业务登记凭证

5. 税务局
   └── 出口退税备案
   └── 获得：出口退税资格

6. 银行
   └── 开设外币账户（如需要）
```

**预计耗时：** 2-4周

**自己办理适合：**
- 时间充裕
- 想省代理费
- 已有公司且想正规运营

---

### 方式二：找代理办理（付费，省事）

**为什么找代理？**
- 流程熟悉，办理快
- 不用自己跑各部门
- 费用不高（1000-2000元）

**怎么找代理？**
- 搜索："城市 + 进出口权代办"
- 阿里1688搜索服务
- 朋友推荐
- 福步论坛

**代理费用：**
| 项目 | 费用 |
|------|------|
| 代理服务费 | 1000-2000元 |
| 海关IC卡 | 约500元（代收） |
| 电子口岸 | 约300元（代收） |

**办理时间：** 1-2周

---

## 三、找代理办理的具体流程

### 第1步：咨询对比

**问代理公司几个问题：**

1. 你们办理多久？（一般1-2周）
2. 包含哪些证件？（要覆盖海关、外汇、电子口岸）
3. 费用多少？还有其他收费吗？
4. 需要我提供什么资料？

**选择要点：**
- 口碑好（福步论坛有评价）
- 价格透明（无隐藏费用）
- 沟通顺畅

### 第2步：提供资料

**通常需要：**

| 资料 | 说明 |
|------|------|
| 营业执照副本 | 复印件 |
| 法人身份证 | 复印件 |
| 股东身份证 | 复印件（如有） |
| 公司章程 | 复印件 |
| 注册地址证明 | 租赁合同或房产证 |
| 公章 | - |

**注意：** 营业执照经营范围需包含"货物进出口"或"进出口贸易"

### 第3步：代理操作

**代理帮你做：**

```
代理工作内容：
├── 准备所有申请表格
├── 跑海关办理备案
├── 办理电子口岸卡
├── 外汇管理局备案
├── 税务局出口退税备案
└── 全部证件交给你
```

### 第4步：领取证件

**你会拿到这些：**

| 证件 | 作用 |
|------|------|
| 海关登记证 | 报关用 |
| 电子口岸法人卡 | 线上报关 |
| 电子口岸操作员卡 | 线上报关 |
| 外汇管理局备案回执 | 收汇用 |
| 出口退税备案 | 退税用 |

## 四、没公司怎么办？（三种解决方案）

### 方案一：注册公司

**流程：**
1. 核名
2. 提交注册资料
3. 领取营业执照
4. 办理进出口权

**费用：**
- 注册公司：免费（自己办）
- 代理注册：1000-2000元
- 进出口权代理：1000-2000元

**适合：** 长期打算、正规运营

---

### 方案二：挂靠外贸代理

**什么是挂靠？**
- 用外贸代理公司的名义出口
- 他们提供抬头、报关、收款
- 你付代理费（1-2%）

**操作流程：**
```
1. 找外贸代理公司
2. 签订代理协议
3. 你负责开发客户、报价
4. 客户付款到代理账户
5. 代理扣除费用后转给你
6. 你国内采购付款给工厂
```

**优点：**
- 不用自己注册公司
- 不用办进出口权
- 立即可以开展业务

**缺点：**
- 代理费（1-2%）
- 客户资源积累在代理名下

**适合：** 试水外贸、兼职起步

---

### 方案三：注册个体工商户

**优点：**
- 门槛低
- 费用少
- 办理快

**缺点：**
- 很多地区不能直接办理进出口权
- 经营范围限制

**适合：** 小规模试水

---

## 五、办理后的下一步

### 1. 开通外币账户

**去银行需要带：**
- 进出口权全套证件
- 营业执照
- 法人身份证
- 公章、财务章

**可以开外币的银行：**
- 中国银行
- 工商银行
- 建设银行
- 农业银行

### 2. 了解出口流程

```
1. 接单
2. 制作发票、装箱单、合同
3. 订舱（货代）
4. 报关（自己或代理）
5. 装柜出海
6. 拿到提单
7. 收汇
8. 结汇
9. 出口退税（如果有）
```

## 六、常见问题

### Q：进出口权办理需要多长时间？

**自己办：** 2-4周
**找代理办：** 1-2周

### Q：没有进出口权能出口吗？

**可以！** 用外贸代理公司挂靠，代理帮你出口。

### Q：进出口权每年要交钱吗？

**不要！** 一次办理，长期有效。但每年要做海关年报（免费）。

### Q：一个人可以办理进出口权吗？

**可以！** 只要有公司（个体户也可以尝试），法人自己跑或委托代理都可以。

### Q：注册公司需要多少钱？

**分情况：**
- 自己办理：0元（工本费可能免）
- 代理注册：1000-2000元
- 地址挂靠：1000-3000元/年（如果没有实际地址）

## 七、行动清单

### 决定办理方式

- [ ] 如果长期打算：自己注册公司 + 办理进出口权
- [ ] 如果试水阶段：先找外贸代理挂靠

### 如果选择代理办理

- [ ] 搜索3-5家代理，对比价格和服务
- [ ] 确定一家，签订协议
- [ ] 提交资料
- [ ] 等待1-2周拿证
- [ ] 开通外币账户
