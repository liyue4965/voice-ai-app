# 新手外贸起步指南（孟村管件版）

---

## 阶段一：准备期（1-2个月）

### 1. 办理基础资质
- [ ] 办理进出口权（找代理公司，费用1000-2000元）
- [ ] 初期不需要自己成立公司，可用外贸代理公司挂靠
- [ ] 代理公司选择要点：
  - 资质齐全（海关、外汇、电子口岸）
  - 靠谱

### 2. 产品学习
- [ ] 了解管件基础知识（规格、材质、标准）
- [ ] 记住核心产品：弯头、法兰、三通、管帽
- [ ] 学会看图纸和参数
- [ ] 熟悉材质：碳钢、不锈钢、合金钢
- [ ] 熟悉标准：GB、ANSI、DIN、JIS

### 3. 找工厂资源
- [ ] 联系孟村本地工厂，获取产品目录和报价
- [ ] 推荐起步合作：有自营出口权的工厂（资质全）
- [ ] 沟通要点：
  - 最小起订量（MOQ）
  - 交货周期
  - 付款方式
  - 是否含税、是否含运费

---

## 阶段二：获客期（3-6个月）

### 获客渠道对比

| 渠道 | 难度 | 成本 | 效果 | 适合阶段 |
|------|------|------|------|----------|
| 阿里国际站 | ⭐⭐ | 中等 | 需投入广告费 | 起步 |
| 中国制造网 | ⭐⭐ | 较低 | 适合工业品 | 起步 |
| 海关数据找客户 | ⭐⭐ | 较低 | 精准但需技巧 | 进阶 |
| 参加展会 | ⭐⭐⭐ | 高 | 效果好但投入大 | 成熟 |
| LinkedIn开发 | ⭐ | 极低 | 长期积累 | 长期 |

### 建议起步方案
> 先用**中国制造网**或**阿里国际站**起步，门槛较低

### 平台操作要点
- [ ] 完善公司介绍（工厂实力、产能、资质）
- [ ] 上传产品图片（高质量工厂实拍）
- [ ] 设置关键词优化
- [ ] 每天坚持更新产品
- [ ] 及时回复询盘（24小时内）

---

## 阶段三：成交期

### 第一单建议
- [ ] 目标市场：优先中东或东南亚（门槛低）
- [ ] 金额控制：几千到一万美金
- [ ] 付款方式：T/T预付30%，余款提单COPY（安全）
- [ ] 选品建议：从常规型号开始，复杂规格后期再碰

### 成交流程
```
1. 询盘接收 → 2. 成本核算 → 3. 报价回复 → 4. 议价沟通
5. 合同签订 → 6. 收取定金 → 7. 下单生产 → 8. 验货发货
9. 收取尾款 → 10. 单据寄送 → 11. 售后服务
```

### 常用付款方式
| 方式 | 优点 | 缺点 | 适用 |
|------|------|------|------|
| T/T 预付 | 安全 | 客户可能有顾虑 | 新客户 |
| T/T 30%+70% | 平衡风险 | - | 一般客户 |
| L/C信用证 | 银行担保 | 手续复杂 | 大客户 |
| D/P | - | 有拒付风险 | 信任客户 |

## 💡 给新手的实在建议

### 1. 别囤货
- [ ] 先询盘，询到订单再找工厂采购
- [ ] 零库存起步，风险最低

### 2. 别垫资
- [ ] 用外贸代理，货物款由客户付
- [ ] 代理费通常1-2%

### 3. 先学习
- [ ] 可以先应聘外贸公司做半年，积累经验再单干
- [ ] 了解外贸全流程：报价、合同、物流、收款

### 4. 产品专一
- [ ] 先只推1-2款产品，别什么都做
- [ ] 专而精更容易获客

### 5. 客户选择
- [ ] 新客户优先做预付
- [ ] 欠款客户需严格审核
- [ ] 记录客户背景调查

## 常见问题

### Q：没有出口权能接外单吗？
> 可以！用外贸代理公司挂靠，他们提供抬头和报关，你负责开发和收款。

### Q：英语不好能做外贸吗？
> 可以！现在很多客户用邮件/翻译软件沟通，前期可以借助工具，重点是把产品搞熟悉。

### Q：需要投入多少钱？
> 平台入驻费 + 广告费（约2-5万起步），代理费1-2%。
