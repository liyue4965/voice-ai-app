# VoiceAI 语音转文字 App - 设计规范

## 1. 项目概述

- **项目名称**: VoiceAI
- **类型**: 跨平台 App (iOS + Android)
- **框架**: React Native + Expo
- **核心功能**: 实时语音录制 + 语音转文字

## 2. UI 设计（完全参考图片）

### 2.1 整体风格
- **主题**: 深色科技风
- **背景**: 深蓝渐变 (#0A0E17 → #1A1F35)
- **主色**: 紫蓝渐变 (#6C5CE7 → #00D9FF)
- **强调色**: 青色 (#00D9FF)

### 2.2 页面结构

```
┌─────────────────────────────────┐
│  ← 语音转文字              👤   │  ← 顶部导航栏
├─────────────────────────────────┤
│                                 │
│      🔴 录音按钮（120px）       │  ← 居中大圆形按钮
│      ~~~ 紫色波纹扩散 ~~~       │  ← 波纹动画
│                                 │
│         ⏱ 00:00:00              │  ← 计时器
│       "点击开始录音"           │  ← 提示文字
│                                 │
├─────────────────────────────────┤
│   🏠       🎙️        👤        │  ← 底部Tab
│   首页     录音       我的      │
└─────────────────────────────────┘
```

### 2.3 颜色系统
- **背景主色**: #0A0E17 (深蓝黑)
- **背景次色**: #1A1F35 (深蓝)
- **渐变起始**: #6C5CE7 (紫色)
- **渐变结束**: #00D9FF (青色)
- **强调色**: #00D9FF
- **录音中**: #FF4757 (红色)
- **文字主色**: #FFFFFF
- **文字次色**: #8B9DC3

## 3. 页面详情

### 3.1 首页 (Tab 1)
- 顶部导航栏：左侧返回(隐藏) + 中间标题 + 右侧头像
- 居中：录音按钮 + 波纹动画 + 计时器
- 底部Tab栏

### 3.2 录音页 (Tab 2) - 同首页

### 3.3 我的 (Tab 3)
- 用户头像区域
- 历史记录列表
- 设置入口

## 4. 功能规范

### 4.1 核心功能
1. **录音** - 点击开始，再次点击停止
2. **波形动画** - 录音时显示波纹动画
3. **计时器** - 显示录音时长
4. **语音转文字** - 使用 Expo Speech 或 Web Speech API

### 4.2 交互
- 点击圆形按钮 → 开始/停止录音
- 点击Tab → 切换页面

## 5. 技术栈

- React Native + Expo
- expo-av (录音)
- expo-speech (语音识别)
- @react-navigation/native (导航)
- lucide-react-native (图标)
